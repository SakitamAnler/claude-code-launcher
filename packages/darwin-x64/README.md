# sakitamanler-ccl-darwin-x64

sakitamanler Claude Code Launcher for macOS Intel

这是 `sakitamanler-ccl-launcher` 的 macOS Intel (x64) 平台包。

## 安装

此包会通过主安装器 `sakitamanler-ccl-launcher` 自动安装，无需手动安装。

## 使用

安装完成后，您可以直接使用 `ccl` 命令：

```bash
ccl                    # 交互式选择 provider
ccl --provider=GLM-4.7 # 指定 provider 运行
```
