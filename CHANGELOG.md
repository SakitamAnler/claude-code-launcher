# 更新日志 (Changelog)

## [v1.1.7] - 2026-01-01

### 🎨 界面优化
- **简化成功框显示** - 移除了复杂的边框装饰，改为简洁的文本提示
  - 之前：`┌─────────────────────────┐  ✓ Claude Code 已安装  └─────────────────────────┘`
  - 现在：`✓ Claude Code 已安装`
- **保留核心信息** - 保持必要信息的清晰展示，去除视觉干扰

### 🔧 功能修复
- **修复返回上一级功能** - 解决了选择"返回上一级"后仍显示同一个 provider 的问题
  - 现在选择"返回上一级"会正确清空 provider 选择，重新显示 provider 选择界面
- **修复 TUI 隔离进程问题** - 移除了子进程相关的 TUI 选择器代码
  - 避免 Windows 环境下的兼容性问题
  - 不再出现 "TUI 隔离进程失败" 警告

### 🛠️ 代码改进
- **优化导入路径** - 修复了开发环境下的导入路径错误
  - 从 `./utils.build.js` 改为 `./utils.js`
- **简化代码结构** - 移除了不再需要的子进程选择器代码
- **统一版本管理** - 确保构建版本与发布包版本一致

### 📦 发布信息
- **win32-x64 包**: `sakitamanler-ccl-win32@1.1.7`
- **installer 包**: `sakitamanler-ccl-launcher@1.1.7`

---

## [v1.1.6] - 2026-01-01

### 🎨 界面美化
- **修复边框对齐问题** - 去掉了多余的装饰线条字符
- **统一视觉效果** - 改善了成功框和信息框的显示效果

### ⚡ 性能优化
- **移除子进程依赖** - provider 和启动模式选择现在都在主进程中直接运行
- **改善交互体验** - 提高了交互式选择的稳定性

---

## [v1.1.0] - 2026-01-01

### ✨ 新功能
- **返回上一级支持** - 在启动模式选择界面可以返回到 provider 选择
- **多级导航** - 支持在程序中进行多级选择和导航

### 🔧 基础架构
- **TUI 选择器** - 实现了基于子进程的 TUI 交互界面
- **版本注入机制** - 实现了构建时版本号注入功能

---

## 安装说明

### 最新版本安装
```bash
npm install -g sakitamanler-ccl-launcher@1.1.7
```

### 卸载旧版本
```bash
npm uninstall -g sakitamanler-ccl-launcher
```
